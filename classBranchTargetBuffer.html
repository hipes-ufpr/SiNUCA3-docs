◆ Query()

void BranchTargetBuffer::Query ( const StaticInstructionInfo *  instruction,
int  connectionID 
)
inlineprivate

Method for Request Query.

Parameters
addressThe fetch address received by a request message.
connectionIDThe ID of the connection that received the request.

When it receives a request, the BTB checks that the address received is valid (if there is an entry in the BTB for it). Then, if it is valid, it checks the corresponding entry and sends a reply message containing the address of the next fetch block and a limited vector containing the prediction bits of the instructions in the current block. If it is invalid, it sends a reply message containing the address of the next sequential block (current address + interleaving factor) and a limited vector with all bits set to 1, assuming that all instructions in the block are predicted to execute.
