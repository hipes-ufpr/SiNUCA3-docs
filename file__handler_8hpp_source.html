52/*
53 * In case of a non standard memory access, the numbers of read and write
54 * operations are stored first, hence the following constant exist, and then
55 * memory accesses itself. It is done that way to save storage space. It
56 * represents the size in bytes designated in the file for the purpose of
57 * storing these numbers.
58 */
59const unsigned long SIZE_NUM_MEM_R_W = sizeof(unsigned short);
60/*
61 * Before each new BBL (basic block) is stored in the static trace, the number
62 * of instructions that make up the block is stored. Thus, the following
63 * constant is the size in bytes designated to store it in the file.
64 */
65const unsigned long SIZE_NUM_BBL_INS = sizeof(unsigned int);
