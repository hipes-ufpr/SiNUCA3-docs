Detailed Description

template<typename MessageType>
class Component< MessageType >

All components shall inherit from this class. The MessageType type parameter defines the message type the component receives from other components. If the component does not receive any message, int can be used as a placeholder. Note that Component<T> is just a wrapper for the underlying mother class Linkable. This is done to centralize the message-passing implementation in a non-template class for optimization reasons. This wrapper allows a nice and type-safe API on top of a single, fast and generic implementation.

Avoiding big types in MessageType is a good idea, because they're passed by value.
