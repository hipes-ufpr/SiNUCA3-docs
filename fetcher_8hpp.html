typedef int FetchBufferEntryFlags

Enum for flags for the fetch buffer of the fetcher.

We use constants and not enums because C++ complains about usingenums as sets (bit-oring them and stuff). It's an int because alignment would make anything occupy more space in the struct anyways so let's use the type that enables us to operate faster on the registers.
