Detailed Description

The Fetcher is a generic fetcher with support to an instruction memory and a predictor. It handles the pay of misspredicton penalties.

It accepts the following parameters:

  • fetch (required): Component<InstructionPacket> from which to fetch instructions.
  • instructionMemory (required): Component<InstructionPacket> to which send the instruction after fetching.
  • fetchSize: The size in bytes to fetch per fetch cycle. Defaults to 0.
  • fetchInterval: Integer amount of cycles to wait before fetching. I.e., fetch every fetchInterval cycles. Defaults to 1.
  • predictor: Component<InstructionPacket> to which send prediction requests.
  • misspredictPenalty: Integer amount of cycles to idle when a missprediction happens.
