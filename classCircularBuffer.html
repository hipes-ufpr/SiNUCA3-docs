Detailed Description

A performant circular buffer.

You may use this as a queue. Use Allocate() to init it.

◆ Allocate()

void CircularBuffer::Allocate ( int  bufferSize,
int  elementSize 
)

Allocates the structure of a Circular Buffer.

Parameters
bufferSizeWhen > 0, sets a limit size, and trying to enqueue more elements will result in an error. When 0, the buffer grows as needed.
elementSizeSets the element buffer. For instance, elementSize = 4 and bufferSize = 4 will allocate a circular buffer of 16 bytes.
