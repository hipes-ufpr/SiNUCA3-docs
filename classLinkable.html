Detailed Description

Do not inherit directly from this class.

This class implements the message-passing of the components in a fast and generic manner, without using templates. For this reason, it's not type-safe, but serves as a generic pointer for any component type. Pointers to Linkable should be avoided, and only used when it's not certain and does not matter which type of message the component receives. It's not possible to send a message directly to a Linkable. The Component<T> wrapper methods should be used.

◆ FinishSetup()

virtual int Linkable::FinishSetup ( )
pure virtual

This method should be declared here so the simulator can send the finish setup message.

This method is called after the config file is and all parameters are set, so to finish any setup required by the component. Non-zero should be returned if any problem occurred (e.g., a required configuration parameter was not provided). The component is responsible for printing a proper error message describing what happened.

Returns
Non-zero on error, 0 otherwise.
